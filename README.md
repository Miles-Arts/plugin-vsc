# plugin-vsc

Este es un proyecto personal creado por Milton Figueredo para aprender a desarrollar plugins para Visual Studio Code y apoyar a la comunidad.

## Licencia
Este proyecto está bajo la licencia MIT.

## Configuración Inicial

### Descargar Node.js
Para descargar Node.js, visita el sitio oficial y sigue las instrucciones de instalación:
[Node.js](https://nodejs.org/)

### Configurar el Generador de Extensiones
Ejecuta los siguientes comandos en tu terminal para instalar las herramientas necesarias:

```bash
npm install -g yo generator-code
```

Este comando instalará Yeoman y el generador de extensiones de Visual Studio Code.
